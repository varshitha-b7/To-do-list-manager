Python To-Do List Manager

A command-line task management application developed using Python to help users organize and track their daily activities efficiently.

Internship Project

This project was developed as part of the CodSoft Python Programming Internship.

Task: To-Do List Application
Internship Domain: Python Programming
Organization: CodSoft

The objective of this project was to build a task management system that allows users to create, view, update, and manage their daily tasks through an interactive menu-driven interface.

Features

Task Management

- Add New Tasks
- View Existing Tasks
- Mark Tasks as Completed
- Delete Tasks
- Track Task Status

User Experience

- Interactive Menu System
- Simple Command-Line Interface
- Easy Task Navigation
- Clear User Feedback

Technologies Used

- Python 3
